# Basic Marketplace

Тестовое задание для VK Tech, Backend Go developer, Intern.

## Как запустить

```bash
docker-compose up --build
```

## Технологии

- Go, Gin, GORM
- PostgreSQL, Redis
- JWT, Docker

## Структура

- cmd/ — точка входа
- internal/ — бизнес-логика, обработчики, инфраструктура
- models/ — основные структуры
